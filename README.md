About
-------

An Android client for the Tempo self-hosted music streaming software package, allowing you to stream your music and other media on any web-accessible device. It was written using Kotlin, and uses the [OkHttp](https://square.github.io/okhttp/) and [Picasso](https://square.github.io/picasso/) open-source libraries by Square for efficient web requests and image processing.

Requires Tempo Server for the hosting of your music and media. [TempoServer on GitHub](https://github.com/chriszilla1123/tempoServer)


Installation
-------

[Google Play](https://play.google.com/store/apps/details?id=net.chilltec.tempo) **[Recommended]**

Build from source:
* Clone the github repository
 `git clone https://github.com/chriszilla1123/tempoAndroid`
* Import into [Android Studio](https://developer.android.com/studio) for installation
	



Requirements
------------
[Tempo Server](https://github.com/chriszilla1123/tempoServer) - The companion server for media hosting.
